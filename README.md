# Gourmet Haven Restaurant

Gourmet Haven is a premium restaurant website featuring a fusion of Fast Food and Authentic Desi Cuisine. This responsive web application showcases the restaurant's menu, ambiance, and allows users to make table reservations.

## Features

-   **Responsive Design:** Built with Bootstrap 5 to ensure a seamless experience across all devices (mobile, tablet, desktop).
-   **Interactive Menu:**
    -   Categorized menu items (Fast Food, Desi Food).
    -   Filter functionality to easily browse specific categories.
-   **Reservation System:**
    -   User-friendly booking form with fields for date, time, guests, and special requests.
    -   Visual feedback upon successful submission (simulated).
-   **Gallery & Ambience:** A visual showcase of the restaurant's interior and atmosphere.
-   **Testimonials:** Customer reviews section.
-   **Contact Information:** Location, phone, email, and an embedded Google Map.
-   **Smooth Navigation:**
    -   Sticky navbar with transparency effects on scroll.
    -   Smooth scrolling for anchor links.

## Technologies Used

-   **HTML5:** Semantic structure.
-   **CSS3:** Custom styling and animations.
-   **Bootstrap 5:** Framework for responsive layout and components (Navbar, Cards, Modals, Forms).
-   **JavaScript (Vanilla):**
    -   DOM manipulation for menu filtering.
    -   Form handling and validation simulation.
    -   Scroll events for navbar styling.
-   **Google Fonts:** Typography (Playfair Display, Lato).
-   **Bootstrap Icons:** scalable vector icons.

## Setup and Usage

1.  **Clone the repository:**
    ```bash
    git clone <repository-url>
    ```
2.  **Open the project:**
    Navigate to the project directory and open `index.html` in your preferred web browser.
3.  **Explore:**
    -   Scroll through the home page.
    -   Click "Menu" to see the dishes and try the filters.
    -   Click "Book a Table" to test the reservation form.

## Project Structure

-   `index.html`: Main structure of the website.
-   `style.css`: Custom styles and overrides for Bootstrap.
-   `script.js`: Logic for interactivity (menu filtering, scroll effects, form handling).
-   `README.md`: Project documentation.
